RPi Image

Raspberry Pi image

The image contains:

  • Raspbian Stretch
  • ROS Kinetic
  • A software package for Clever

The latest version of the image is available for downloading from GitHub in section Releases.

The stable and supported version of the image is the release marked as Latest release.

Installing the OS image on a MicroSD card

To install the image, you can use the [Etcher] utility (https://etcher.io).

Image version

The version of the installed image may be found in file /etc/clever_version:

cat /etc/clever_version
